# Recipes App

This is an example app to see and upload recipes. You can log in with your email or Google account.

## Features

- Home
- Detail
- Login
- My profile
- Add recipe
- Favorites
- Offline mode

## Tech and libs

- Kotlin
- Retrofit
- LiveData
- Dagger for DI
- Firebase realtime database for remote
- Room for cache
- Glide and Lottie
- MVVM pattern
- Modularization
- Unit testing with Mockito and jUnit 4
- UI testing with Espresso
